# Decompose by psychic structure, not astrological factor

The agent ensemble slices the chart by **structures of the psyche** (Shadow,
Persona, Anima/Animus, parental complexes, the wound, vocation, …) — each agent
reading the *whole* chart through one structure — and a single strong
**interpreter** agent synthesizes their outputs into one holistic report from the
individuation / Self vantage. We rejected the obvious factor-based decomposition
(one agent per planet, house, or aspect) because it atomizes the chart into
parallel cookbook summaries that never speak to each other; structure-agents are
natively *in tension* (the Shadow agent sees exactly what the Persona agent
hides), which supplies the dialogue of perspectives the depth aim requires.

## Consequences

- The agent roster encodes a specific psychology (Jungian), not the chart's
  mechanics. Changing schools means re-rostering, not re-plumbing.
- Coverage of every placement is guaranteed not by per-placement agents but by
  the deterministic **chart brief** every agent reads.
- The **Self** is not a peer agent — it is the synthesizing vantage of the
  interpreter. (Whether to also run an explicit Self/individuation agent that
  *feeds* the interpreter is an open roster question, not a change to this
  decision.)
